# ippo-data-analysis-ML-project

Notebook-based Google Colab project.

## Main file
- `ipp_waste.ipynb`
  - starts from your original analysis cells,
  - then continues with missing advanced cleaning,
  - completes KPI analysis,
  - and adds full machine learning model training/comparison for waste prediction.

## Usage
1. Open `ipp_waste.ipynb` in Google Colab.
2. Upload your Excel file.
3. Set `DATASET_ROOT` / `DATASET_PATH` cell value.
4. Run all cells in order.
5. Download generated files from `/content/outputs`.
